포인터의 원리를 이해하는 것이 중요하다.
그래서 포인터의 원리를 이해하는 데 도움이 되는 문제를 아래에 제시하였다.
문제를 천천히 읽어본 다음 문제의 요구사항대로 프로그램을 작성해보자.
다소 입맛에 맞지 않는 문제라 하더라도 여러분에게 많은 도움이 될 것이다.

문제 1
아래의 예제 실행시 변수와 포인터 변수의 관계를 그림을 그려서 설명해보자.
또한 출력의 결과도 예상해보자.
int main(void)
{
    int num = 10;
    int * ptr1 = &num;
    int * ptr2 = ptr1;

    (*ptr1)++;
    (*ptr2)++;
    printf("%d \n", num);
    return 0;
}
