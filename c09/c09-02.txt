다음은 프로그램 사용자가 입력하는 값을 누적하여 그 합계를 출력하는 예제다.

#include <stdio.h>
int total = 0;

int AddToTotal(int num)
{
  total += num;
  return total;
}
int main(void)
{
  int num, i;
  for(i = 0; i < 3; i++)
  {
    printf("Input %d: ", i +1);
    scanf("%d", &num);
    printf("Accumulate: %d \n", AddToTotal(num));
  }
  return 0;
}

위의 예제에서는 함수 AddToTotal에서의 사용을 목적으로 전역변수 Total을 선언하였는데, 이를 static 변수로 대체해보자.
단, static 변수로의 대체과정에서 main 함수의 변경은 없어야 하며, 실행 결과도 동일해야 한다.
